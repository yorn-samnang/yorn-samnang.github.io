# Portfolio Website

Welcome to my portfolio website! This website is a showcase of my projects, skills, and professional experiences. Explore the site to learn more about me and my journey. 

## Features

- **About Me**: Learn about my background, expertise, and interests. 
- **Projects**: A curated collection of my best work with detailed descriptions and repository links. 
- **Education**: My academic journey and achievements. 
- **Skills**: Technical and soft skills with detailed descriptions. 
- **Contact**: Easily reach out to me through social media links. 
- **Responsive Design**: Enjoy a seamless experience across desktops, tablets, and smartphones. 
- **Clean & Minimal**: Professional design with smooth transitions and hover effects. ✨

## Technologies Used

- **HTML5**: Semantic markup structure 
- **CSS3**: Pure CSS with Flexbox and Grid layouts (No frameworks!) 
- **JavaScript**: Interactive navigation and mobile menu 
- **Fonts**: Google Fonts (Roboto & Poppins) 
- **Icons**: Font Awesome 6.0 
- **Design**: Custom gradients, animations, and background patterns 

## Project Highlights

 **Pure CSS** - No Bootstrap or CSS frameworks  
 **Favicon** - Custom SVG favicon with brand initial  
 **Background Images** - Subtle dot pattern background  
 **Multiple Fonts** - Roboto for body, Poppins for headings  
 **External Stylesheet** - All styling in `style.css`  
 **Organized Assets** - Images properly structured in `/assets/images/`  
 **Mobile-First** - Fully responsive with hamburger menu  
 **Professional Footer** - Compact and minimal design
